# Branding Data Schema

User's brand identity and voice settings.

## Fields
- `brand_voice` - Tone of communication (professional, friendly, etc.)
- `target_audience` - B2B, B2C, specific demographics
- `tagline` - Company tagline/slogan
- `value_propositions` - Key value props
- `colors` - Brand colors (if provided)
- `fonts` - Brand fonts (if provided)
- `logo_url` - Logo URL

## Source
- Onboarding flow
- User updates via chat
- Website analysis (inferred)
