# Button Patterns

## SYNTAX
Simple: `$$**emoji Text**$$`
With description: `$$**emoji Title|Description here**$$`

## EXAMPLES
```
$$**📧 Create Newsletter**$$
$$**✅ Yes, continue**$$
$$**❌ No, cancel**$$
$$**⏭️ Skip for now**$$
$$**👔 B2B Decision Makers|CTOs, CEOs, Directors**$$
$$**💬 Start Chat with Newsletter Agent**$$
$$**➕ Add Another Assistant**$$
```

## RULES
1. One button per line
2. Always include emoji first
3. Keep title concise (2-5 words)
4. Use action verbs (Create, Edit, Send)
5. Use `|` for descriptions when needed

## DO NOT USE
- Markdown links `[text](url)`
- Raw URLs
- Multiple buttons on same line
- Buttons without emojis
