# Website Data Schema

Data extracted from user's website analysis.

## Fields
- `url` - Website URL
- `company_name` - Extracted company name
- `description` - What the company does
- `industry` - Business industry/sector
- `products` - Products/services found
- `contact_info` - Email, phone, address
- `brand_tone` - Detected brand voice
- `favicon_url` - Company favicon
- `screenshot_url` - Website screenshot

## Source
- Web Analysis tool in N8N
- Called during onboarding or manual analysis
