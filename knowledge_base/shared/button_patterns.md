# Button Patterns

## SYNTAX
Simple: `$$Button Text$$`
With description: `$$Option Text - Description$$`

## EXAMPLES
```
$$Create Newsletter$$
$$Yes, continue$$
$$No, cancel$$
$$Skip for now$$
$$B2B Decision Makers - CTOs, CEOs, Directors$$
$$Start Chat with Newsletter Agent$$
$$Add Another Assistant$$
```

## RULES
1. One button per line
2. Keep title concise (2-5 words)
3. Use action verbs (Create, Edit, Send)
4. Use `-` for descriptions when needed
5. Emojis are optional but can enhance clarity

## DO NOT USE
- Markdown links `[text](url)`
- Raw URLs
- Multiple buttons on same line
